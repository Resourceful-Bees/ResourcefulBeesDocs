# **Mutation Data**

## **Mutation Types** (Required)
***

### **NONE**

This is used when a bee has no mutation. ex. `"mutationType": "NONE"`


### **FLUID_TO_FLUID**

This is used when a bee mutates a fluid block into another fluid block. ex. `"mutationType": "FLUID_TO_FLUID"`


### **BLOCK_TO_FLUID**

This is used when a bee mutates a fluid block into another fluid block. ex. `"mutationType": "FLUID_TO_FLUID"`


### **FLUID_TO_BLOCK**

This is used when a bee mutates a fluid block into another fluid block. ex. `"mutationType": "FLUID_TO_FLUID"`


### **BLOCK_TO_BLOCK**

This is used when a bee mutates a fluid block into another fluid block. ex. `"mutationType": "FLUID_TO_FLUID"`
